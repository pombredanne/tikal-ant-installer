Apache Ant 1.7.0 Manual

This is the manual for version 1.7.0 of Apache Ant. If your version of Ant (as verified with ant -version) is older or newer than this version then this is not the correct manual set. Please use the documentation appropriate to your current version. Also, if you are using a version older than the most recent release, we recommend an upgrade to fix bugs as well as provide new functionality.

 

This set of documents is reposted from the Ant1.7.0 downloadable manual so users of AntInstaller have the exact documents that are relevant to the specific Ant version used in the current AntInstaller download.
